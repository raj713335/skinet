# skinet

### 1. Clone the Repository:

```
git clone https://github.com/raj713335/skinet
```


### 2. Run the .NET(5.0) Backend Application:

```
cd skinet
dotnet clean
dotnet build skinet.sln
cd API
dotnet watch run --skinet
```

### 3. Run the Angular Frontend Application:

```
cd skinet/client
npm install
ng serve
```
